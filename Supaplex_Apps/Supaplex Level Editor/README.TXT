                Редактор уровней к игре Supaplex
               Автор: Ландайс Александр Николаевич
                             2000 год.

                             Введение

Этот редактор я написал будучи на летник каникулах после первого курса.
Нас как раз обучили программированию на языке Си. Я решил написать что-
нибудь серьезное. В этот период я был увлечен игрой Supaplex. К этому
времени я прошел игру вдоль и поперек, делал кое-какие изменения в
файлах и даже начал разбираться в структуре файла в котором храняться
уровни. Я понял что смогу написать редактор уровней и сделал это.
Получилась невзрачная программа без графического интерфейса, но
позволяющая изменять уровни, количество алмазов, которые надо собрать
и т.д.

                          Ключевые клавиши

F1 - Help
F2 - Save levels.dat (сохранение)
F3 - Load levels.dat (загрузка)
F4 - Change infotrons (количество алмазов, которые надо собрать)
F5 - Copy level (копировать уровень в память)
F6 - Change level number (выбрать уровень)
F8 - Paste level (вставить уровень из памяти)
F9 - Create level.lst (создать файл с именами уровней)
PgUp/PgDown - Change level number (предыдущий/следующий уровень)
Ins - Reset cursor (обнулить курсор)
Del - Reset level  (очистить уровень)
-/+ - Change cursor (изменить курсор)
Esc - Quit (выход)


                           Использование

Данной программе для редактирования уровней необходим файл с
уровнями "levels.dat". Для запуска редактора необходимо запустить
файл "e.bat". Далее, используя описанные выше клавиши, редактируйте.


                            Примечание

Используйте этот редактор в полноэкранном режиме. Т.к. я использую
специальный шрифт, в котором прорисовал все требуемые элементы игры.

Четвертый уровень в прилогаемом файле "levels.dat" я создал без
редактора уровней используя только шестнадцатиричный редактор в
программе Dos Navigator. Первый уровень может пройти только самый
опытный игрок. Этот уровень придумал я сам и утверждаю что его
можно пройти.
