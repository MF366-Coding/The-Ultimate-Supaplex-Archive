What you can do with this program:
With this program, you can edit .dat level files from Supaplex.
--------------------------------------------------------------------

Using the program:

1) Start the program
2) Click 'Load file' to load a .dat file
3) Select a file that you want to edit
4) Now you can change the level data.
5) Click 'Save level' to save the current level.

In the next version, you can edit the X and Y positions of a port.
